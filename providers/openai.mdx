# OpenAI Provider

The OpenAI provider allows you to automatically track all OpenAI API calls in your agent.

## Installation

```bash
pip install playgent openai
```

## Basic Usage

Use `wrap_openai()` to wrap your OpenAI client and the `@record` decorator on your agent functions:

```python
from openai import OpenAI
from playgent import record, wrap_openai

client = wrap_openai(OpenAI())

@record
def my_agent(user_input: str):
    response = client.chat.completions.create(
        model="gpt-4",
        messages=[{"role": "user", "content": user_input}]
    )
    return response.choices[0].message.content
```
